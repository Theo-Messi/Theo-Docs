### 自动获取

- 登录[阿里云盘](https://www.aliyundrive.com/drive/)后，控制台粘贴

```sh
copy(JSON.parse(localStorage.token).refresh_token); console.log(JSON.parse(localStorage.token).refresh_token);
```
