## 脚本特性

- 集成 clash，不依靠客户端
- 落地机 ip 检测,检测复用 ip
- 导出为 Excel

## 下载脚本

```sh
git clone git@github.com:quzard/netflix-all-verify.git
```

## 使用方法

```sh
./netflix-all-verify
#[输入clash的订阅地址]
```
