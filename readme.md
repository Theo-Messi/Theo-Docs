## 介绍

Theo-Docs 是一个基于 [VitePress](https://vitepress.dev/) 搭建的文档网站，用于记录和分享前端开发相关的知识。

## 特性

- 基于 [Vitepress](https://vitepress.dev/) 搭建
- 支持 Markdown 语法
- 支持主题定制
- 支持代码高亮
- 支持搜索功能
- 支持评论功能
- 支持导航栏和侧边栏
- 支持页面布局定制
- 支持页面样式定制
- 支持页面内容定制
- 支持页面内容自动生成
- 支持页面内容自动更新
